## 🔬 Technical Details

### Commission-Based CPC Formula
```python
max_cpc = (avg_price × commission_rate / 100 × 0.5) / clicks_per_conversion
```
- Ensures profitability at keyword level
- CPC never exceeds 50% of commission per conversion
- Different clicks/conversion for different keyword types:
  - Brand keywords: 20 clicks/conversion (3.6% CVR)
  - Product keywords: 30 clicks/conversion (2.6% CVR)
  - Long-tail phrases: 50 clicks/conversion (3.4% CVR)

### Organic Search Estimation
```python
organic_clicks = paid_clicks × 2.0  # Conservative 2x multiplier
organic_cvr = paid_cvr × 0.8        # 80% of paid CVR
seo_cost = paid_budget × 0.3        # 30% of paid budget
```
- Timeline: 3-12 months to reach full potential
- Month 1-3: 20-30% of estimated traffic
- Month 4-6: 50-70% of estimated traffic
- Month 7-12: 80-100% of estimated traffic

### Vietnamese Number Parsing
Handles multiple formats:
- Thousand separator: `1,000,000` → `1000000`
- Thousand separator: `1.000.000` → `1000000`
- Decimal separator: `4,5` → `4.5`
- Decimal separator: `4.5` → `4.5`

Logic:
1. Multiple separators → thousand separator (remove all)
2. Both comma and dot → last one is decimal
3. Single comma with ≤2 digits after → decimal
4. Single dot with ≤2 digits after → decimal
5. Otherwise → thousand separator

### Number Parsing Issues
The parse_number() function handles Vietnamese formats automatically.
If issues persist, check the format:
- Use comma for decimal: `4,5`
- Use comma for thousands: `1,000,000`
- Do not mix formats in same number
